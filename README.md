<h1>Naiden</h1>
A multi-purpose Discord bot used in the Portland Public Schools server.

<h2>Features</h2>
Completed features have their checkboxes filled.

- [ ] Automatic voice-channel creation ("Lair" system)
- [ ] Counting game
- [ ] Reaction Roles
- [ ] Statistics/Leveling
